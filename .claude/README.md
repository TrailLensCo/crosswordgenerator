# Claude Code Project Settings

This directory contains configuration files for Claude Code integration.

## Files

- **config.json** - Project-specific settings and preferences for Claude Code
  - Project metadata and description
  - Coding standards and conventions
  - Development workflow preferences
  - AI assistance preferences
  - File patterns to ignore

## Usage

These settings help Claude Code understand:
- The project structure and conventions
- Copyright and licensing requirements
- Coding style preferences (Black, PEP 8)
- Documentation standards
- Branch protection and PR workflow

Claude Code will automatically reference these settings when providing assistance in this repository.
